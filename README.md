# ossflow-studio

SPA React de gestión del ecosistema OSSFlow.

## Tech Stack

- React 18 + Vite
- Tailwind CSS
- Zustand (state)
- React Router

## Desarrollo

```bash
cd ossflow-frontend
npm install
npm run dev
```

Por defecto se conecta a `http://localhost:8000` (ossflow-api). Para apuntar a otro backend, ajustar `vite.config.js`.

## Producción

```bash
cd ossflow-frontend
npm run build
```
